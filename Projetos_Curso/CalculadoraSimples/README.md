# Calculadora com While

Este projeto foi desenvolvido durante o curso como forma de praticar
lógica de programação em Python.

## Objetivo
Treinar:
- Estruturas de repetição (`while`)
- Estruturas condicionais (`if / elif / else`)
- Validação de entradas do usuário
- Tratamento de erros com `try / except`

## Funcionalidades
- Recebe dois números do usuário
- Recebe um operador matemático (+, -, *, /)
- Valida se os números são válidos
- Valida se o operador é permitido
- Impede divisão por zero
- Permite continuar ou encerrar o programa

## Aprendizados importantes
- Entendi a importância de validar entradas do usuário
- Aprendi como funciona a lógica do `and` e do `or`
- Usei `try / except` para evitar que o programa quebre com entradas inválidas

## Observações
Este projeto tem foco em aprendizado e prática de lógica.
Sugestões de melhoria e feedbacks são bem-vindos
