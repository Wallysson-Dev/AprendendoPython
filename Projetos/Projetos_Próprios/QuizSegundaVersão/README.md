# Quiz em Python

Este projeto é uma versão refeita do meu primeiro quiz em Python, desenvolvido
pouco mais de **1 mês** depois da versão inicial, com o objetivo de testar minha
**evolução em na programação**.

---

## Objetivo

Praticar e reforçar conceitos fundamentais da linguagem **Python**, como:

- Entrada de dados (`input`)
- Estruturas de repetição (`while`)
- Estruturas condicionais (`if / elif / else`)
- Uso de listas para organizar dados
- Manipulação de strings (`strip`, `lower`)
- Geração de valores aleatórios (`random`)
- Controle de fluxo (`break`, `continue`)
- Uso de contadores e operador módulo (`%`)

---

## Como funciona

- As perguntas são armazenadas em **listas**
- A cada rodada, uma pergunta é escolhida **aleatoriamente**
- O usuário digita a resposta
- O programa verifica se a resposta está correta
- A pontuação é atualizada conforme **acertos e erros**
- Perguntas já feitas são removidas para evitar repetição
- A cada certo número de perguntas, o usuário pode decidir se deseja continuar
- A cada certo número de perguntas, o usuário pode decidir se dejesa limpar a tela
- Ao final, o programa exibe o **resultado do jogo**

---

## Aprendizados

- Organização de código utilizando listas
- Uso de índices para manter dados relacionados sincronizados
- Evitar repetição de perguntas removendo itens da lista
- Validação e normalização da entrada do usuário
- Uso de contadores para controle do jogo
- Aplicação do operador módulo (`%`) para ações periódicas
- Melhor entendimento do fluxo do programa
