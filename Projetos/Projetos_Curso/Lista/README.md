# Lista Interativa em Python

Projeto desenvolvido a partir de uma ideia do curso, com foco em praticar
lógica de programação e manipulação de listas em Python.

## Objetivo
Treinar conceitos fundamentais da linguagem, como:
- Listas (`list`)
- Estrutura de repetição (`while`)
- Estruturas condicionais (`if / elif / else`)
- Validação de entradas do usuário
- Uso de índices com `enumerate`
- Controle de fluxo com `continue` e `break`

## Funcionalidades
- Inserir itens em uma lista
- Listar todos os itens com seus índices
- Apagar itens da lista pelo índice
- Verificar entradas inválidas do usuário
- Impedir erros como índices fora do intervalo da lista
- Menu interativo em loop até o usuário sair

## Aprendizados
- Entendi melhor como funcionam listas e índices
- Aprendi a validar a entrada do usuário antes de usar no código
- Aprendi a evitar erros comuns, como acessar índices inexistentes
- Pratiquei lógica com menus e repetição usando `while`

## Observações
Este projeto tem foco em aprendizado e prática de lógica.
O código ainda pode ser melhorado, mas representa bem minha evolução
no uso de listas e controle de fluxo em Python.
