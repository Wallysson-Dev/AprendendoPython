# Quiz em Python

Este foi meu primeiro projeto em Python, criado para testar minhas habilidades
de lógica de programação.

## Objetivo
Praticar conceitos básicos de programação, como:
- Entrada de dados (`input`)
- Estruturas condicionais (`if / else`)
- Operadores lógicos (`or`)
- Controle de pontuação
- Manipulação de strings (`lower`)

## Como funciona
- O programa apresenta perguntas ao usuário
- O usuário digita a resposta
- O programa verifica se a resposta está correta
- A pontuação aumenta em caso de acerto
- Em caso de erro, o usuário pode perder pontos

## Aprendizados
- Aprendi a trabalhar com validação simples de respostas
- Usei operadores lógicos para tratar mais de uma resposta correta
- Aprendi a controlar pontos usando variáveis
- Entendi melhor o fluxo de decisões com `if` e `else`

## Observações
Este projeto tem foco em aprendizado e prática de lógica.
Ainda é um projeto simples, mas representa o início da minha jornada
na programação.
